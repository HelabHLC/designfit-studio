# ARBE λ* Branding

## Canonical platform name

**ARBE λ***

Descriptor: **The Reference Platform for Spectral Colour Intelligence**

Short form in ordinary prose: **ARBE**. The short form does not replace the canonical platform name in titles, formal architecture documents or first-use descriptions.

## Product name

**ARBE DesignFit Studio** is the first reference application built on the ARBE λ* Platform.

## Scientific term

**ARBE λ*_V2** denotes the normative physical ordering parameter and must not be used as a synonym for the whole platform.

## Module names

- ARBE Core
- ARBE Atlas
- ARBE AtlasFit
- ARBE MixLock
- ARBE Spectral Scissor
- ARBE Metamerism Gate
- ARBE Measured
- ARIA

## Domain rule

A web domain is deployment configuration only. It is never a platform identity, dataset identity, reference identity or audit identity. No domain may be placed in canonical branding until ownership and deployment authority are confirmed.

## Prohibited ambiguity

Do not describe ARBE λ* as merely a colour app, colour generator, Pantone replacement or production-certification service.
