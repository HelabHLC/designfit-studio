# Recipe Intelligence Evidence Package v1

WP06.8 combines one verified Recipe Decision Support Report with one verified Recipe Evidence Assessment Report for every ranked candidate.

The package enforces a shared HLC target, exact candidate coverage, valid source integrity, deterministic ordering and a canonical SHA-256 package digest.

The package records evidence and review guidance only. It does not identify a best recipe, certify equivalence, approve a formulation or grant production release.

Only an existing Hxxx_Lxxx_Cxxx record is an ARBE identity.
