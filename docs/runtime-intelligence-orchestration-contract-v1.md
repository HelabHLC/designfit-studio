# Runtime Intelligence Orchestration Contract v1

WP07.1 defines the deterministic runtime sequence connecting Reference Gateway, Spectral Intelligence, optional Recipe Intelligence and final Runtime Report Binding.

Required stages stop on invalid or missing evidence. Recipe Intelligence is conditional, but any skip must be recorded explicitly as evidence.

Only an existing `Hxxx_Lxxx_Cxxx` record is an ARBE identity. HEX, Lab, images and external colour standards remain requests.

The contract controls sequencing only. It does not create scientific evidence, certify spectral or visual equivalence, confirm root cause, approve a recipe or grant production release.
