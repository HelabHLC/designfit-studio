# ARBE Measured v0.1 — Initial Acquisition Set

Status: acquisition target list, not a validated spectral library.

## Purpose

Build a compact, legally clean and scientifically documented set of real artist colours for AtlasFit, MixLock and Spectral Scissor validation.

Selection principles:

- prefer single-pigment products;
- avoid duplicate pigment chemistry in the first pass unless the comparison is intentional;
- record manufacturer, line, article number, batch and declared pigment index;
- do not treat a product name as an ARBE colour identity;
- bind each measured sample to an HLC atlas reference only after measurement;
- preserve raw instrument files and checksums.

## Exclusions from the mandatory first pass

Cadmium, cobalt and other heavy-metal pigments are not required for v0.1. They may be added later under a separate handling and safety protocol.

## Product selection rule

For each target, choose one real product only after confirming:

1. pigment index is declared;
2. product line and article number are stable;
3. batch number can be recorded;
4. material can be applied reproducibly;
5. no manufacturer restriction prevents publication of ARBE's own measurements.

## Measurement design

Each product should be prepared at minimum as:

- one controlled full-strength drawdown;
- one masstone measurement set;
- three measurement positions;
- optional tint with PW6 in a documented ratio;
- optional transparent layer over a standard black/white substrate.

The first public release should contain only records passing the ARBE Measured v0.1 protocol and publication gate.
