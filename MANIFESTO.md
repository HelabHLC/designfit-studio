# ARBE λ* Manifesto

**Colour is communication. Reference is evidence.**

ARBE λ* provides a reference-bound framework for spectral colour analysis, atlas identity, pigment-aware formulation and evidence-based validation.

It separates colour requests from colour identity, candidates from reference locks, and numerical similarity from physical evidence.

The recipe is the candidate. The curve decides. The reference lock makes it valid. The metamerism gate qualifies the lock.

Applications are built on the platform; they do not redefine its scientific foundation.
