# ARBE λ* Vision

ARBE λ* is not another colour application. It is the reference platform on which colour applications can be built.

The platform separates:

- colour requests from colour identity;
- candidates from reference locks;
- numerical similarity from spectral evidence;
- public code from controlled data;
- scientific definitions from application behaviour.

Its purpose is to provide a shared scientific and computational foundation for colour analysis, atlas binding, pigment-aware formulation and evidence-based validation across applications and industries.
